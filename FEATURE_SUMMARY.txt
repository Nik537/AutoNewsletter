📦 WHAT'S NEW
───────────────────────────────────────────────────────────────────────────

1. ✅ YouTube URL Input
   • New "YouTube URL" tab in the upload interface
   • Paste any YouTube video link
   • No manual downloading required

2. ✅ Automatic Video Download
   • Uses yt-dlp (professional YouTube downloader)
   • Downloads best quality MP4 automatically
   • Shows download progress (5-15%)

3. ✅ Seamless Integration
   • Downloaded videos processed like uploaded files
   • Same AI analysis and newsletter generation
   • Same Slovenian output

4. ✅ User-Friendly Interface
   • Toggle between "Upload File" and "YouTube URL"
   • Clean, modern input form
   • Progress tracking for downloads

───────────────────────────────────────────────────────────────────────────
🎯 HOW TO USE
───────────────────────────────────────────────────────────────────────────

1. Open http://localhost:3000
2. Click "🔗 YouTube URL" button
3. Paste YouTube link (e.g., https://www.youtube.com/watch?v=...)
4. Click "Process Video"
5. Wait 3-10 minutes (download + processing)
6. Download your Slovenian newsletter!
