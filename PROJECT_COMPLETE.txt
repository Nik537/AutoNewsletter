🚀 GETTING STARTED
────────────────────────────────────────────────────────────────────────────────

1. Install Prerequisites:
   • Python 3.9+
   • Node.js 16+
   • FFmpeg

2. Setup:
   ./setup.sh

3. Configure:
   Edit backend/.env with ANTHROPIC_API_KEY

4. Launch:
   ./start.sh

5. Use:
   Open http://localhost:3000
   Drag video → Wait → Download
